# cnn_particlefilter  





# Requirements  
cnn_particlefilter requires the following to run:  

- Caffe  
- Python2.7  
- Opencv3.x  
# Usage  
Install python libraries:  
`$ pip install scikit-image`  

Clone:  
`$ cd caffe/examples`  
`$ git clone git clone https://github.com/Rehide/cnn_particlefilter.git`  

Run:  
`$ cd cnn_particlefilter/python`  
`$ python tracking.py`
